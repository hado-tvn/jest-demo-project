# jest-demo-project

## Install npm dependencies
```shell
 npm install
 ```

## How to run
1. unit test

```shell
npm test tests/unit/task27.test.js
```

2. integration test

```shell
npm test tests/integration/integration.test.js
```

3. e2e test for browser

```shell
npm test tests/e2e/browser.test.js
```
